**Badge** — small status / metadata pill (SLA state, priority, tier, counts).

```jsx
<Badge tone="success">Within SLA</Badge>
<Badge tone="danger" variant="solid">High</Badge>
<Badge tone="cord">Enterprise</Badge>
```

Tones map to the semantic + pendant palette: `neutral | cord | accent | success | warning | danger | info`. `variant="soft"` (tinted) default; `solid` for emphasis.
